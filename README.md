# Memulai training

Eksekusi training melalui terminal

```bash
python train.py --model mobilenetv3_large --epoch 50
```

Sesuaikan argument model dengan model yang tersedia di bawah ini.

MODEL:
- mobilenetv3_small
- mobilenetv3_large
- proposed_large_16
- proposed_large_32
- proposed_small_16
- proposed_small_32
